About the Website

Our project is called the Speech-Emotion Distinguisher. This project is what we made for our Senior Capstone project over the Summer of 2023. What this application does is use a convolutional Neural Network embedded into a Python (Flask) backend. The CNN is trained on a large speech emotion dataset and analyzes certain features to determine the closest emotional match. The highest accuracy we were able to achieve was around 70% for the eight-class classification.
